Options

 duration - the duration of appear fade effect (default value : 1.0)

 delay -  the time in seconds for which each slide is displayed (default value: 4.0)

 controls - includes buttons (next, previous, play and stop) to control the slideshow (default value: false)

 random - puts slides in random (default value: false)

 slideshow - automatic start of the slideshow (default value: true)
